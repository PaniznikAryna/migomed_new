Холестерин – это жироподобное вещество, необходимое для нормального функционирования организма. Он является важным компонентом клеточных мембран, участвует в синтезе половых гормонов, витамина D и желчных кислот, необходимых для переваривания жиров.

В медицинском центре “Мигомед” доступны следующие анализы для проверки уровня холестерина:

  • обмен холестерина расширенный (код услуги - 11225);
  • биохимический скрининг функции печени (код услуги - 11092);
  • контроль здоровья (минимальный) (код услуги - 11211).
